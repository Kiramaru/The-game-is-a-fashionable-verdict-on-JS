Модный приговор

Для начала работы приложения необходимо загрузить фотографию лица. Затем вы сможете передвигать аксессуары с
помощью мышки, для этого перетащите выбранный аксессуар на фотографию лица. Затем уже находящиеся
на лице аксессуары можно двигать мышкой нажав по ним 2 раза, отцепить их можно нажав 1 раз. Изменять пропорции можно
только фотографии лица. Поворачивать можно только аксессуары с помощью клавиш R и L. Так же с помощью стрелок
можно двигать выбранные аксессуары. Категорически запрещается перетаскивать аксессуары за границы фотографии,
вы их убьете.
